    现在，您无需传票或搜查令即可找到该 IP 地址的位置。

    一家名为MaxMind的公司维护着一个包含地球上每个 IP 地址位置的数据库，包括 GPS 坐标、区号、邮政编码和国家/地区。该数据库不是典型的关系数据库格式，而是平面文件。MaxMind 收取 370 美元的站点许可费和 90 美元/月（或 1360 美元/年）的数据库更新费用。他们的软件有一个漂亮的前端，可以轻松查询数据库，即使是 Windows 或 Mac 用户也可以管理。

    MaxMind 还提供了该数据库的免费开发者版本，无需任何软件或工具来读取它。虽然比商业版稍微差一点，但价格肯定是合适的。我们只需要一个程序来查找 IP 的位置即可读取这些数据。

    两位程序员 Jennifer Ennis 和 T. Williams 开发了一个名为 pygeoip 的小型 Python 脚本，并在 GPL 许可下发布，使我们能够输入 IP 地址并输出此关键信息。我认为本教程是不言自明的，但如果您想在深入学习之前更新您的 Python 技能，请查看我们的 Python 教程。

    第 2 步：下载数据库

    现在我们需要从 MaxMind 下载数据库，我们可以通过键入以下内容来获取它。

    kali > wget -N -q http://geolite.maxmind.com/download/geoip/database/GeoLiteCity.dat.gz

    然后我们需要解压它。

    kali> gzip -d GeoLiteCity.dat.gz

    现在让我们通过列出目录来检查数据库是否就位。

    kali > ls -alh GeoLiteCity.dat

    接下来，我们需要安装 Python 脚本来读取数据库 pygeoip。我们可以通过键入以下内容来下载它。

    kali > wget http://pygeoip.googlecode.com/files/pygeoip-0.1.3.zip

    然后，解压。

    kali > 解压 pygeoip-0.1.3.zip

    接下来我们需要将一些设置工具下载到 pygeoip 目录中。

    卡利 > cd /pygeoip-0.1.3

    kali > wget http://pypi.python.org/packages/2.5/s/setuptools-0.6c11-py2.5.egg

    现在让我们移动，然后构建和安装设置工具。

    kali > mv setuptools-0.6c11-py2.5.egg setuptools-0.7a1-py2.5.egg

    kali > python setup.py 构建

    kali > python setup.py 安装

    我们需要将数据库移动到 pygeoip 目录，以便脚本可以访问它而无需使用完整路径。

    kali > mv GeoLiteCity.dat /pygeoip-0.1.3/GeoLiteCity.dat
